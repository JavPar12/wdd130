Portfolio

Explore some of my notable projects in programming:

  • Project 1: WWR Website

    In this project, I utilized HTML and CSS, to create an interactive website that promotes the thrilling experience of white water rafting. The website design focused on conveying the excitement and adrenaline of this extreme activity, using vibrant colors and impactful visual elements. Features such as booking forms, image galleries, and customer testimonials were implemented to provide a comprehensive view of the experience. The project was developed following best practices in web development, ensuring scalability, security, and a seamless user experience.

    To see the Proyect, visit: WWR Proyect

  • Project 2: Barbershop Website

    For this project, I developed a modern and stylish website for a barbershop. The design focused on creating a masculine and welcoming atmosphere, with clean lines and bold typography. The website includes information about the services offered, pricing, and booking options. The implementation involved HTML and CSS, to create interactive elements such as image sliders and appointment forms. The website was optimized for responsiveness, ensuring a seamless experience across different devices.

    To see the Project, visit: Barbershop Project
